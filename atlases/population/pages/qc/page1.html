What this page shows
scaffold · panel ships when MODULE_2B emits
The orientation page. One row per sample, sortable by any column. This is where any reader starts when picking up the atlas — get oriented to the cohort, then drill into the topical pages (heterozygosity, diversity, inbreeding) as questions arise.
Hover any row → mini-card with summary stats. Click sample ID → per-sample deep-dive sub-view (planned). Filter bar at top by family, broodline, ancestry-Q top class, F_ROH percentile.
Planned columns
field schema · awaiting MODULE_2B JSON layer
field source module description
sample_id samples.ind Canonical sample identifier (CGA-prefixed in this cohort).
family_id pedigree manifest Family-of-origin from breeding records; matches MODULE_2B family overlay.
broodline pedigree manifest Hatchery broodline (parent stock identity); typically 3-5 broodlines per cohort.
mean_coverage MODULE_QC mosdepth Mean WGS coverage across callable genome (target ~9×).
callable_fraction MODULE_QC callable BED Fraction of genome inside callable regions (post-mask).
genome_het 02_heterozygosity/04_summary Genome-wide heterozygosity (per-sample θ_W or H_obs).
F_ROH MODULE_3 ngsF-HMM Genome-wide inbreeding from runs of homozygosity (ROH-derived F).
ancestry_Q_top MODULE_2B NGSadmix Dominant ancestry component at K=3 (or chosen K).
natora_status MODULE_2B NAToRA Kinship-pruning status (kept / removed). Two structure rounds: all 226 + 81 unrelated.
qc_pass MODULE_QC summary Boolean — passes coverage / missingness / contamination thresholds.
